HTML gets its name as Hyper Text as it will allow the users to click on links of the document to view other documents. This is what is called as Hyper Links.
We use <a> to create a Hyperlink in HTML Documents.
Each link will have attributes like href that tells the location of the file that U want to view. It will be usually relative paths. U can have additional Styles for the Anchor Tag. Attributes like target = _blank will allow the file to be viewed in another tab of the browser instead of the same Tab.
Tags can also have IDs so that U can move within the same document.

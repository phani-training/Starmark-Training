About Forms

Forms are the way to take inputs from the user for UR application. Forms work with Server side technologies for storing, processing and rendering the data based on the content that is sent to the server.
A server, A web server is a software installed in large machines that will process requests made by the users across the internet to provide web content. This server will host the Web Application and will handle all kinds of Requests using HTTP protocol.
We have server side technologies like ASP, ASP.NET, JSP, PHP that will facilitate the processing of the Requests made over HTTP.
Every Http Request is returned as HTTP Response from the server. Response could a Page that the user has requested or a data.

Forms send the content(Inputs given by the user) to the server in 2 ways: GET and POST. Get sends the info thru URL. This is not safe. Another way is to POST the data using BODY Content of the HTTP. Forms will have tags like input Tag that will take input from the users. By default, all input tags are Edit boxes.

  • Form Tag is used to create a form in a Web Page.
  • method attribute is used to tell the browser how the content is transfered to the server
  • action attribute is used to define the function that the server uses to process the posted data
  • Inputs given by the user can also be processed using Client side Programming languages like JavaScript
  • All HTML Documents are programmatically represented as objects in a tree view manner. We call that as DOM(Document Object Model).
  • Using the DOM functions, we can get access to any element of the document by using a method called getElementByxxx
  • U can get access to elements based on Id, Tags or CSS Classes.
  • DOM methods can be called in the HTML document under <script> tag.

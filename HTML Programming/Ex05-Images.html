Images usage in HTML

  1. Images can be created using <img> Tags
  2. Its attribute src will specify the location of the image.
  3. Other attributes like width and height could define the size of the Image
  4. If the image could not be loaded for various reasons, U can use an attribute called alt to provide an alternate text to display in the page
  5. It is recommended to create a Folder of images and use them in the application. Also beware of the ©copyright issues of the images that U want to use in UR application.
